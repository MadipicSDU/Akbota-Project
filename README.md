# Freelance Platform for Students

A frontend-only pet project — a freelance marketplace UI for students to find projects, connect with clients, and manage work.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui (Radix UI)
- React Router v7
- Mock data (no backend)

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Pages

- `/` — Dashboard
- `/projects` — Browse projects
- `/projects/:id` — Project details
- `/projects/create` — Post a project
- `/profile` — User profile
- `/messages` — Messaging
- `/payments` — Payment history
- `/pricing` — Pricing plans
- `/admin` — Admin panel
- `/auth/login` — Login
- `/auth/register` — Register
